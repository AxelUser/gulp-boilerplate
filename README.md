# gulp-boilerplate
My boilerplate for gulp-based projects to quickly bootstrap new ones.

Structure:

[**root**]  
|  
--[**gulp-tasks**] *//Tasks for Gulp :)*  
|  
--[**public**] *//Final application*  
|  
--[**src**] *//All sources*  
|  
----[**assets**]  
|  
----[**js**]  
|  
----[**less**]  
|  
----[**views**]
